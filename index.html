더이상 기술적으로 구현 불가능한 플랫폼은 없습니다.
개발자와 사용자가 함께 구축하고자 하는 플랫폼을 이해하며 개발 전 단계에서 사용자가 참여하여 쉽게 쓰일 수 있는 UI, 꼭 필요한 기능을 구현합니다.

플랫큐브의 리깅 프로그램은 건설 산업 현장에서 쓰이는 리깅 프로그램을 모바일 환경에서 기술적으로 구현하였습니다.
더 나아가 정비 솔루션까지 함께 제공할 예정입니다.

Mobile Environment

Crane Rigging Program Maintenance Solution

크레인 리깅 및 유지보수 프로그램

크레인 리깅 프로그램은 이동식 크레인을 사용하는 환경에서 반드시 써야하는 작업 시뮬레이션 프로그램입니다. 기존의 환경은 데스크탑에서만 있는 캐드 프로그램과 크레인 제원표를 함께 보면서 작업 시뮬레이션을 설정하였습니다. 이를 한번에 묶어 모바일 앱으로 만들어 작업시간을 획기적으로 단축시켰습니다.

Miner Monitoring System

코인 마이닝 모니터링 시스템

당시 대규모의 코인 마이닝팜의 개발요청에 따라 3000대 규모의 마이닝 서버를 연결하는 네트워크 시스템을 구축하였습니다. 다양한 종류의 마이너의 온도, 채굴 속도, 오류 상황과 같은 다양항 정보를 실시간으로 모니터링 할 수 있는 시스템을 개발하였습니다. 각각의 마이너는 24시간 원격으로 모니터링 및 제어할 수 있으며 최고 수준의 안정성을 갖는 보안 시스템을 구축하였습니다 비전문가도 기기의 상태를 정확하고 빠르게 체크할 수 있고 실시간으로 채산성을 확인할 수 잇는 UI/UX가 장점인 시스템입니다.

  • Date: 2017
  • Client: Lian GTC
  • Category: WebBase Admin Page
